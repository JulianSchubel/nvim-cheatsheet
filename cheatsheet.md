# Neovim Cheatsheet

## Normal mode
- `h j k l` — Move cursor
- `w b e` — Word motions
- `0 ^ $` — Line start / first char / end
- `gg G` — Top / bottom of file
- `dd yy p` — Delete / yank / paste
- `u / <C-r>` — Undo / redo

## Insert mode
- `i a o` — Insert before / after / new line
- `I A O` — Smart insert
- `<Esc>` — Back to normal

## Visual mode
- `v V <C-v>` — Char / line / block select
- `y d p` — Yank / delete / paste

## Windows
- `<C-w>s / <C-w>v` — Split / vsplit
- `<C-w>h j k l` — Move between windows
- `<C-w>q` — Close window

## Buffers
- `:ls` — List buffers
- `:bnext / :bprev` — Next / previous buffer
- `:bd` — Delete buffer

## Search
- `/ ?` — Search forward / backward
- `n / N` — Next / previous match
- `* / #` — Word under cursor

## Help
- `:help {topic}` — Help for topic

---
Press `q` or `<Esc>` to close
